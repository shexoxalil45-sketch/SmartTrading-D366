Mobile quickstart:

1) Install Flutter SDK (stable >= 3.2)
2) From project root:
   flutter pub get
   flutter run -d <device>

3) Build release APK:
   flutter build apk --release

Backend:
1) cd backend
2) python -m venv venv
3) source venv/bin/activate
4) pip install -r requirements.txt
5) python app.py
